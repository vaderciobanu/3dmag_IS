# 3dmag_IS
3DMAG STL Library este o aplicație REST care gestionează modele 3D (STL-uri) și resurse de printare precum imprimante, materiale și culori. Sistemul oferă operații CRUD pentru toate entitățile și funcționalități precum descărcări, rating-uri și compatibilitate între modele și imprimante. Aplicația folosește .NET, SQLite și Docker Compose.
